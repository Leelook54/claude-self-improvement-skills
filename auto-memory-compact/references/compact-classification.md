# Compact Classification

## 分类法

### keep
仍然有效、被引用且有用。保留在当前位置。

### move-to-topic
详细内容应放在单独的 topic 文件中。提取并引用。

### merge-duplicate
覆盖相同内容的重复条目。合并为一个。

### archive-stale
过时、不再适用或被取代的条目。归档。

### reject-low-value
琐碎、显而易见的单次条目，无持久价值。丢弃。

### contradiction-fix
互相矛盾的条目。标记待解决。

### promotion-candidate
可能值得提升到 CLAUDE.md 或 skill 的条目。标记待批准。

### skill-candidate
值得提取为 skill 的流程或工作流。标记待创建 skill。

### project-specific
仅与一个项目相关的条目。移至项目级记忆。

### global-behavior-candidate
跨项目行为规则。考虑提升到 GLOBAL memory 并获得批准。
