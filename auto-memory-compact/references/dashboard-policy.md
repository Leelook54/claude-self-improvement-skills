# Dashboard Policy

## Purpose
定义 SELF_IMPROVEMENT_DASHBOARD.md 的内容、行数、更新规则。

## Target File
~/.claude/memory/SELF_IMPROVEMENT_DASHBOARD.md

## Rules
- Dashboard is for human orientation only.
- Dashboard is not active rules.
- Keep under 80 lines when possible.
- Hard limit: 120 lines.
- Link to latest reports, pattern index, promotion index, compact status.

## Required Sections
- Current system version
- Last review
- Last compact
- Inbox count
- Errors count
- Active patterns count
- Proposed promotions count
- Skill candidates count
- Active warnings
- Archived notes
- Recommended next actions
- Do-not-treat-as-rules warning

## Do Not Include
- long review report text
- raw error logs
- full transcript quotes
- detailed archive contents
- rules that should belong in SKILL.md or references
